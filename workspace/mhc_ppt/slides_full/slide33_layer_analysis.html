逐层分析发现

惊人发现：不稳定性始于第 0 层！

通过逐层分析 Amax，Taylor 发现 HC 的不稳定性从输入层（第 0 层）就开始，而不是在深层逐渐累积。原因：第 0 层直接处理原始 Embeddings，没有 LayerNorm 的保护。

HC 逐层稳定性 (红=不稳定)

Layer 0 → Layer 31

mHC 逐层稳定性 (绿=稳定)

Layer 0 → Layer 31

关键洞察：第 0 层的输入混合矩阵是"金丝雀"——它最先显示不稳定迹象。如果你只能监控一个地方，就监控第 0 层。mHC 在所有层级、所有训练步数中均呈现绿色（稳定），完美验证了双随机约束的有效性。
