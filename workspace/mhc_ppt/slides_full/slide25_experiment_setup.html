Part 5: 实验验证

DeepSeek 官方实验设置

模型架构

类型: MoE (受 DeepSeek-V3 启发)

专家数: 64 个路由专家 + 2 个共享专家

激活专家: 每 token 激活 8 个

测试规模

3B 参数 (基础验证)

9B 参数 (中等规模)

27B 参数 (主要关注)

HC/mHC 配置

扩展率 n: 4 (4条并行信息流)

Sinkhorn 迭代: 3 次

精度: BF16 混合精度

训练数据

数据量: 2T tokens

来源: DeepSeek 内部数据集

实验重点关注 27B 参数模型，这是 HC 问题最严重的规模，也是验证 mHC 有效性的关键。
